# PaperCorpus2Skill Agent MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a working local agent and CLI that generate a Markdown Skill Pack from PDF and Markdown paper corpora.

**Architecture:** A small Python package exposes `PaperCorpus2SkillAgent` as the reusable core. The CLI delegates to the agent; later Web UI code can do the same.

**Tech Stack:** Python 3.12-compatible standard library first, optional PyMuPDF for PDF parsing, pytest for tests.

## Global Constraints

- Support PDF and Markdown only for the first implementation.
- Support recursive folder input.
- Support local Zotero storage/WebDAV-synced PDFs when files exist on disk.
- No direct remote WebDAV authentication in this version.
- Keep the core pipeline reusable by CLI and future Web UI.
